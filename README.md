## How to Run

1. Clone the repository

```
git clone https://github.com/your-username/cryptchat.git
```

2. Install dependencies

```
cd cryptchat
npm install
```

3. Start the server

```
python app.py
npm start
```

4. Open the app in your browser

```
http://localhost:3000/
```
